````md
# 🛒 Gerador de QR Code para E-commerce

Projeto desenvolvido durante bootcamp da DIO com foco em Node.js, geração de QR Codes e organização de aplicações por camadas.

## 📌 Sobre o Projeto

Este projeto consiste em um gerador de QR Code para e-commerce desenvolvido utilizando Node.js.

A aplicação foi criada com o objetivo de praticar:
- Estruturação de projetos backend
- Organização de código por camadas
- Uso de dependências externas
- Manipulação de pacotes NPM
- Geração dinâmica de QR Codes

## ⚙️ Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
````

### Acesse a pasta do projeto

```bash
cd nome-do-projeto
```

### Instale as dependências

```bash
npm install
```

```bash
node src/index.js
```
